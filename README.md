# DynaSync

Dynasync es una aplicación móvil centrada a los emprendedores que requieren una
herramienta de gestión en el cual su jerarquía principal se divide en proyectos y en tareas.
Esta aplicación permite también registrar y gestionar los pagos que se hagan con respecto
al emprendedor y gestionar al personal él mismo con la posibilidad de llevar un
seguimiento sobre las tareas asignadas y las fechas finales de objetivos.
